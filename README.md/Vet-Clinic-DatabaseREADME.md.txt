# 🐾 Vet Clinic Database

This project is a complete SQL-based system to manage a veterinary clinic. It covers animals, owners, visits, medications, vets, appointments, and history tracking.

## 📁 Project Structure
- **DDL/**: Contains all table creation scripts.
- **DML/**: Contains insert scripts for fake data.
- **Functions/**: Scalar and table-valued functions for business logic.
- **Views/**: Views used for reporting and simplification.
- **Triggers/**: Triggers for data tracking and validation.
- **Stored_Procedures/**: Procedures for insert/update/delete and advanced logic.

## 📊 ERD
The project is based on a well-structured ERD that supports:
- One-to-many and many-to-many relationships
- Total participation
- Weak entities 
- Proper constraints and normalization

## 🔍 Business Questions
We answered various business questions using:
- Complex joins
- Grouping and aggregation
- Functions and triggers
